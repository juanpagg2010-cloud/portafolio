# Portafolio Juan Pablo Barragan Cortes

Portafolio personal creado con HTML5, CSS3 y JavaScript vanilla.

## Deploy en Render

1. Sube este proyecto a un repositorio de GitHub.
2. En Render, crea un nuevo **Static Site** o usa el Blueprint del archivo `render.yaml`.
3. Configuracion manual si Render la pide:
   - Build Command: `echo "No build step required"`
   - Publish Directory: `.`
4. Render publicara el sitio y te dara una URL `onrender.com`.

## Archivos principales

- `index.html`: estructura del sitio.
- `styles.css`: estilos, responsive y animaciones.
- `script.js`: menu movil, animaciones al hacer scroll y glow del cursor.
